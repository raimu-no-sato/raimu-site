# 来夢の里 WordPress セットアップ手順書
# ファイルをアップロードしてサイトを完成させる手順

---

## 同梱ファイル一覧

```
raimu-child-theme/
├── style.css        ← 子テーマのデザインファイル（全CSS入り）
├── functions.php    ← 子テーマの機能ファイル
└── import.xml       ← ページ・記事の一括インポートファイル
```

---

## STEP 1｜子テーマをZIPにまとめる

**Windowsの場合：**
1. `raimu-child-theme` フォルダを右クリック
2. 「送る」→「圧縮 (zip 形式) フォルダー」を選択
3. `raimu-child-theme.zip` ができる

**Macの場合：**
1. `raimu-child-theme` フォルダを右クリック
2. 「"raimu-child-theme"を圧縮」を選択
3. `raimu-child-theme.zip` ができる

---

## STEP 2｜WordPressに子テーマをインストール

1. WordPress管理画面にログイン
2. 左メニュー「外観」→「テーマ」
3. 「新しいテーマを追加」→「テーマのアップロード」
4. `raimu-child-theme.zip` を選択してアップロード
5. 「インストール済みのテーマを有効化」をクリック

⚠️ **注意：** Cocoon親テーマが先にインストールされている必要があります。
まだの場合は先に Cocoon を検索してインストールしてください。

---

## STEP 3｜WordPressインポーター プラグインのインストール

1. 左メニュー「プラグイン」→「新規追加」
2. 「WordPress Importer」で検索
3. インストール → 有効化

---

## STEP 4｜import.xml でページ・記事を一括作成

1. 左メニュー「ツール」→「インポート」
2. 「WordPress」の「インポーターを実行」をクリック
3. `import.xml` を選択してアップロード
4. 投稿者を「admin」（または自分のアカウント）に割り当て
5. 「送信」をクリック
6. 完了メッセージが出たら成功！

✅ インポート後に作成されるもの：
- 固定ページ 7ページ（トップ・施設・サービス・お菓子・採用・作業・お問い合わせ）
- 投稿記事 6本（ブログ記事3本・採用記事3本）
- 投稿カテゴリー 5つ

---

## STEP 5｜各ページにHTMLコードを貼る

インポート直後のページは「HTMLを貼ってください」という案内文が入っています。
以下の手順で実際のコードに差し替えてください。

```
固定ページ → 各ページを編集
→ カスタムHTMLブロックの内容を
  対応するファイルのコードに差し替え
→ 「更新」で保存
```

| ページ名 | 使うファイル | 場所 |
|---|---|---|
| トップページ | wp_top_page.html | ファイル全体 |
| 施設について | wp_all_pages.html | 「1. 施設について」 |
| サービス内容 | wp_all_pages.html | 「2. サービス内容」 |
| お菓子のご注文 | wp_sweets_shop.html | 上半分（①ページHTML） |
| 採用情報 | wp_recruit_page.html | 「①固定ページ」 |
| 作業依頼 | wp_all_pages.html | 「5. 作業依頼」 |
| お問い合わせ | wp_all_pages.html | 「6. お問い合わせ」 |

---

## STEP 6｜トップページとして設定

1. 左メニュー「設定」→「表示設定」
2. 「ホームページの表示」→「固定ページ」を選択
3. ホームページ：「トップページ」を選択
4. 投稿ページ：「お知らせ（新規作成）」を選択
5. 「変更を保存」

---

## STEP 7｜ナビゲーションメニューを設定

1. 左メニュー「外観」→「メニュー」
2. 「新しいメニューを作成」→名前：「グローバルナビ」
3. 左側の固定ページから以下を追加：

```
施設について
サービス内容
お菓子のご注文
採用情報
作業依頼
お問い合わせ  ← 「ボタン」クラスを追加
```

4. 「メニューの位置」→「グローバルナビゲーション」にチェック
5. 「メニューを保存」

---

## STEP 8｜Contact Form 7 のフォームを作成

以下の3種類のフォームを作成してください。

### フォーム①：共通お問い合わせフォーム
ファイル：`wp_all_pages.html` 下部のCF7コメント参照

### フォーム②：お菓子注文フォーム
ファイル：`wp_sweets_shop.html` 下部のCF7コメント参照

### フォーム③：採用応募フォーム
ファイル：`wp_recruit_page.html` 下部のCF7コメント参照

フォーム作成後、各ページのショートコード `★フォームID★` を実際のIDに差し替えてください。

## STEP 9｜Cocoon設定でブランドカラーを設定

1. 「Cocoon設定」→「全体」
2. メインカラー：`#e67e22`（オレンジ）
3. テキストリンク色：`#c0392b`（赤）
4. 変更を保存

## よくあるトラブル

**Q: 子テーマのアップロードでエラーが出る**
A: Cocoon親テーマが未インストールの可能性があります。先にCocoonをインストールしてください。

**Q: インポート後にページが表示されない**
A: パーマリンク設定をリセットしてください。
「設定」→「パーマリンク」→何も変えずに「変更を保存」をクリック。

**Q: CSSが反映されない**
A: ブラウザキャッシュをクリア（Ctrl+Shift+R）。またはWordPressのキャッシュプラグインのキャッシュを削除。

**Q: フォームのメールが届かない**
A: 「WP Mail SMTP」プラグインをインストールして、ConoHaのSMTP設定を行ってください。
ConoHa管理画面 → メール → SMTPホスト名・ポートを確認。
